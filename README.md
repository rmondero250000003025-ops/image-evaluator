This repository contains handwritten student essays and explores how Large Language Models (LLMs) can be applied to evaluate students' written work.

##  Project Overview
The goal of this project is to demonstrate how LLM-based systems can assist in assessing handwritten essays by:
- Converting handwritten text into digital format (OCR)
- Analyzing grammar, structure, and content quality
- Providing automated scoring and feedback
- Comparing human evaluation vs AI evaluation

##  AI Evaluation Approach
This project explores the use of Large Language Models (LLMs) to:
- Evaluate student writing based on predefined rubrics
- Generate feedback for improvement
- Support teachers in reducing manual grading workload

##  Repository Contents
- Scanned handwritten essays (images/PDF)
- Sample outputs from LLM evaluation
- Documentation of the evaluation process

##  Objective
To study how AI can assist in educational assessment, specifically in grading handwritten student essays more efficiently and consistently.

##  Note
This project is for educational and research purposes only.
